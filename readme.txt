Enables google analytics integration on all pages.

== Description ==

This plugin Easy Web Analytics Integration adds JavaScript tracking code for Google Analytics.
Login into google analytics ,Setup your property here for tracking your website.
And simply copy javascript tracking code from google analytics website property

For more information visit:
(http://www.google.com/analytics)

== Installation ==

1. Upload `googleanalyticsadvanced` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click Settings > Easy Web Analytics Integration
3. Copy and paste Tracking Code into Google Analytics TextArea


== Frequently Asked Questions ==

= How to get google analytics code  =

Go to google analytics.Sign In ,Search javascript analytics code for your property
